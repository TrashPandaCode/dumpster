# Time

The _Time_ node provides the current game time as well as the delta time. On game pause the time node will pause as well and resume on game resume.

## Properties

- **Time**: The current game time in seconds. This value is updated every frame.
- **Delta Time**: The time in seconds since the last frame. This value is updated every frame.
