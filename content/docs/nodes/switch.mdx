# Switch

The Switch node selects and outputs one of two input values, _True_ or _False_, based on a boolean control input.

## Properties

- **Result**: The output value determined by the _bool_ input. It returns either the _True_ or _False_ input.
- **bool**: The boolean control input. If the value is 0, it's treated as false; any other value is considered true.
- **False**: The input value that is output when _bool_ is false.
- **True**: The input value that is output when _bool_ is true.
